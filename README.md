
# 🤖 AI Customer Support Chatbot

A smart, AI-powered customer support chatbot built for small businesses. Instantly answers customer queries 24/7 without human intervention.

## 🔴 Live Demo
👉 [Try it here](https://ai-support-bot--vijaysandeep200.replit.app)

## ✨ Features
- 💬 Real-time AI responses powered by Claude API
- 🎨 Clean, modern chat interface
- ⚡ Instant replies — no waiting
- 🔧 Fully customizable for any business
- 📱 Works on mobile and desktop

## 🛠️ Tech Stack
- Python / Flask
- Anthropic Claude API
- React + Vite (Frontend)
- PostgreSQL (Database)
- Deployed on Replit

## 💼 Use Cases
- E-commerce stores
- Restaurants & cafes
- Service businesses
- Startups & agencies
